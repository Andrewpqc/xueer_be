# xueer_be

    the backend of xueer::学而后台
  
## 0. build
### virtual environment

    $ virtual venv
    $ source venv/bin/activate
    
### install extensions

    $ pip install -r requirement.txt (--no-cache-dir)
    
### test database
#### create Role

    $ chmod 777 data.sh
    $ ./data.sh
    >> Role.insert_roles()

#### fake data

    >> User.generate_fake()
    >> Courses.generate_fake()
    >> quit()

    $ password:
    $ confirm:

### run

    python manage.py run (--help)
